# Weather App

A responsive weather application built using HTML, CSS, and JavaScript that fetches real-time weather data from the OpenWeatherMap API. Users can search for any city worldwide and view current weather conditions through a simple and intuitive interface.

## Features

* Search weather by city name.
* Display current temperature.
* Show humidity levels.
* Display wind speed.
* Dynamic weather icons based on current conditions.
* Error handling for invalid city names.
* Responsive design for desktop and mobile devices.
* Fast and lightweight implementation using vanilla JavaScript.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/weather-app.git
```

### Navigate to the Project Directory

```bash
cd weather-app
```

### Open the Application

Simply open `index.html` in your preferred browser.

## API Configuration

1. Create a free account on OpenWeatherMap.
2. Generate an API key.
3. Replace the API key in `script.js`:

```javascript
const apiKey = "YOUR_API_KEY";
```

## Example Usage

1. Enter a city name (e.g., London, Tokyo, Mumbai).
2. Press the search button.
3. View the current weather information instantly.
